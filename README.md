# 🧩 チュートリアルメーカー

Webサイト向けのインタラクティブなガイドツールをブラウザだけで作成できるスタンドアロンHTMLアプリです。
サーバー不要・インストール不要。`tutorial-maker-standalone.html` を開くだけで使えます。

---

## 概要

作成したチュートリアルはJSONとしてエクスポートでき、対象のWebページに埋め込みコードとして追加するだけで動作します。
ツールチップ表示・要素ハイライト・クリック待ち・入力待ちの4種類のステップに対応しています。

```
┌─────────────────────────────┐
│  チュートリアルメーカー        │  ← ステップを組み立てるエディタ
└──────────────┬──────────────┘
               ↓ エクスポート（埋め込みコード）
┌─────────────────────────────┐
│  対象のWebサイト              │  ← TutorialPlayer が動作
└─────────────────────────────┘
```


| 操作 | 説明 |

## 各ペインの説明

### プレビューペイン

対象のHTMLファイルをiframeで表示し、チュートリアル編集内容をリアルタイムで確認できるペインです。

- **「📂 HTMLを開く」**: HTMLファイルを選択（CSSファイルも同時に選択可能）
- **「↺ リセット」**: iframeを最初から再生し直す
- **リサイズ**: 右端のハンドルをドラッグして幅を調整（200px〜）
- **リアルタイム更新**: ステップを追加・削除・編集するたびにiframe内のチュートリアルが自動更新される

> CSSファイルも読み込む場合は、HTMLファイルとCSSファイルを一緒に選択してください。CSSの`<link href="...">` 参照が自動的にBlob URLに置き換わり、スタイルが適用された状態で確認できます。

### パレット

チェーンに追加するステップの種類を選ぶパネルです。クリックするとチェーン末尾にステップが追加されます。

| アイコン | 種類 | CSSセレクタ | 説明 |
|---------|------|------------|------|
| 💬 | メッセージ | 不要 | ツールチップにテキストを表示するだけのステップ |
| 🔦 | ハイライト | 必須 | 指定要素を青枠でハイライトしながらメッセージを表示 |
| 👆 | クリック待ち | 必須 | 指定要素がクリックされるまで次に進まない |
| ⌨️ | 入力待ち | 必須 | 指定要素への入力またはフォーカス離脱で次に進む |

### チェーン

作成したステップが実行順に並ぶパネルです。

- ステップをクリックするとプロパティパネルで編集できます
- ▲/▼ボタンでステップの順番を入れ替えられます
- ✕ボタンでステップを削除できます

### プロパティパネル

選択中のステップの内容を編集するパネルです。

| フィールド | 説明 |
|-----------|------|
| ラベル | チェーン上での識別名（省略可） |
| メッセージ | ユーザーに表示するガイドテキスト |
| CSSセレクタ | 操作対象の要素を指定する（ハイライト・待ちステップのみ） |

CSSセレクタはDevToolsで要素を右クリック →「Copy > Copy selector」で取得できます。

## HTML解析モーダル

対象HTMLを貼り付けまたはファイルで読み込むと、ボタン・入力欄・リンクなどのインタラクティブな要素を自動検出します。検出された要素をクリックするとCSSセレクタが現在選択中のステップに自動セットされます。

**検出対象タグ:** `button`, `input`, `select`, `textarea`, `a[href]`, `[role="button"]`

**HTMLのセット方法:**
1. テキストエリアに直接ペースト
2. 「📂 ファイルを開く」でHTMLファイルを選択
3. プレビューペインでHTMLを開いていれば、モーダルを開いた時点で自動セット

---

## エクスポートモーダル

### 埋め込みコードタブ

チュートリアルプレイヤーのコードとステップ定義を1つにまとめたHTMLスニペットを出力します。
対象サイトの `</body>` 直前に貼り付けるだけでチュートリアルが動作します。

```html
<!-- チュートリアルPlayer（インライン） -->
<script>
  /* TutorialPlayer のコード */
</script>

<!-- チュートリアル定義 -->
<script>
TutorialPlayer.init({
  "id": "tutorial-1",
  "name": "チュートリアル名",
  "steps": [ ... ]
});
</script>
```

### JSONタブ

ステップ定義のみをJSONで出力します。独自システムへの組み込みや管理に使えます。

## プロジェクト管理

### 自動保存

以下の操作のたびにブラウザのlocalStorageに自動保存されます。ページを閉じても内容は保持されます。

- ステップの追加・削除・並び替え
- プロパティの編集（300msのデバウンス）
- プロジェクト名の変更

### 起動時の復元

前回開いていたプロジェクトが存在する場合、起動時に自動的にそのエディタを開きます。

### データ形式

localStorageキー: `tm_projects`（プロジェクト配列）、`tm_last_project_id`（最後に開いたID）

```json
{
  "id": "proj_1234567890",
  "name": "テトリス操作ガイド",
  "description": "",
  "createdAt": 1234567890000,
  "updatedAt": 1234567890000,
  "steps": [...],
  "nextId": 5
}
```

## チュートリアルプレイヤー

エクスポートされたコードで動作するプレイヤーの仕様です。

### 初期化

```javascript
TutorialPlayer.init({
  id: 'tutorial-1',
  name: 'チュートリアル名',
  steps: [
    { id: '1', type: 'message',    message: 'ようこそ！', selector: '' },
    { id: '2', type: 'highlight',  message: 'ここをご覧ください', selector: '#target' },
    { id: '3', type: 'wait-click', message: 'ボタンをクリックしてください', selector: '#btn' },
    { id: '4', type: 'wait-input', message: 'テキストを入力してください', selector: '#input' },
  ]
});
```

### UI構成

- **ツールチップ**: 画面右下または要素近くに表示（320px幅）
- **オーバーレイ**: 対象要素以外を半透明の暗幕で覆う4分割方式
- **プログレスバー**: 現在のステップ数と進捗を表示
- **スキップ**: 5秒後に「スキップ」ボタンが表示される
- **モバイル対応**: 画面幅480px以下で画面下部に固定表示

## 使い方の流れ

1. `tutorial-maker-standalone.html` をブラウザで開く
2. 「＋ 新規プロジェクト」でプロジェクトを作成
3. **「🔍 HTML 解析」** に対象サイトのHTMLを貼り付けて要素を検出（または手動でセレクタを記入）
4. パレットからステップを追加してプロパティを設定
5. **「👁 プレビュー」** → 「📂 HTMLを開く」でリアルタイム確認
6. **「⬇ エクスポート」** → 埋め込みコードを対象サイトの `</body>` 直前に貼り付け

---

## 動作環境

モダンブラウザ（Chrome / Edge / Firefox / Safari）で動作します。
